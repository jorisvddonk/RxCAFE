# image

Image-related annotations for generated images.

## image.description

| Property | Value |
|----------|-------|
| Type | `string` |

Human-readable description of a generated image. Used as caption for image chunks.

## image.width

| Property | Value |
|----------|-------|
| Type | `number` |

Width of the generated image in pixels.

## image.height

| Property | Value |
|----------|-------|
| Type | `number` |

Height of the generated image in pixels.
